# External Mines Programmator

[![Deploy to GitHub Pages](https://github.com/murasamadsp/External-Mines-Programmator/actions/workflows/deploy.yml/badge.svg)](https://github.com/murasamadsp/External-Mines-Programmator/actions/workflows/deploy.yml)
[![Live Demo](https://img.shields.io/badge/Live%20Demo-Click%20Here-blue)](https://murasamadsp.github.io/External-Mines-Programmator/)

Веб-приложение для визуального программирования ботов для игры "Mines". Этот инструмент позволяет создавать и настраивать автоматизированные сценарии поведения в игре с помощью интуитивно понятного интерфейса.

## О проекте

**External Mines Programmator** — это мощный веб-инструмент, разработанный для создания и управления программируемыми ботами в игре "Mines". Проект предоставляет удобный интерфейс для визуального программирования, что позволяет пользователям без глубоких знаний в кодировании создавать сложные автоматизированные сценарии.

### Ключевые особенности

-   **Визуальный редактор**: Интуитивно понятный интерфейс для создания программ с помощью перетаскивания блоков.
-   **Гибкость и мощность**: Создавайте сложные алгоритмы и стратегии для ботов.
-   **Простота использования**: Не требует навыков программирования для начала работы.
-   **Современный стек технологий**: Создано с использованием Vite, что обеспечивает быструю разработку и высокую производительность.

## Начало работы

Для начала работы с проектом, следуйте этим инструкциям:

### Требования

-   [Node.js](https://nodejs.org/) (версия 14.x или выше)
-   [npm](https://www.npmjs.com/) или [yarn](https://yarnpkg.com/)

### Установка

1.  **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/murasamadsp/External-Mines-Programmator.git
    ```

2.  **Перейдите в директорию проекта:**

    ```bash
    cd External-Mines-Programmator
    ```

3.  **Установите зависимости:**

    ```bash
    npm install
    ```

### Запуск

Для запуска проекта в режиме разработки выполните команду:

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере, чтобы увидеть приложение.
