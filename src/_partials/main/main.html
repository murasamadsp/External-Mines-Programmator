Программатор Mines

Інструкція:

  • Виберіть дію з палітри та натисніть на клітинку сітки для розміщення
  • Для імпорту: вставте Base64 код програми у поле "Import Program" та натисніть "Import"
  • Для експорту: натисніть "Export to Base64" для отримання Base64 коду
  • Натисніть "Validate" для перевірки програми
